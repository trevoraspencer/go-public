# clean-go-repo

Status: stable

A minimal Go project used to verify go-public audit passes on a clean repository.

## Requirements

- Go 1.22+

## Install

```bash
go mod download
```

## Configuration

No configuration required.

## Usage

```bash
go test ./...
```

## Security

No secrets or credentials are stored in this repository.
